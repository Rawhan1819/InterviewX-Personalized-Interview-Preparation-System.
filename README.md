# InterviewX - Personalized Interview Preparation System

InterviewX is a professional, AI-powered platform designed to revolutionize interview preparation. Built with a Multi-Agent system (Resume Agent, Interview Agent, Coding Agent, and more), it provides a deeply adaptive and data-driven experience.

## 🚀 Key Features
- **🏴‍☠️ Luffy AI Interviewer**: A custom 3D avatar (Luffy in a Suit) with real-time NLP-driven conversation and gestures.
- **📄 AI Resume Analyzer**: In-depth ATS scoring and actionable keyword improvements for specific job roles.
- **💻 Integrated Coding Environment**: Monaco-based IDE with automated evaluation for DSA and system design problems.
- **📊 Adaptive Performance Tracking**: Improvement trends (e.g., `5.5 → 6.8 → 7.2`) and detailed qualitative session summaries.
- **🏢 Company-Specific Prep**: Targeted preparation modules for top-tier tech companies.
- **🛣️ AI-Driven Roadmaps**: Personalized career paths based on skill gap analysis.

## 🏢 Academic Branding
The platform is tailored for **Vignan's Foundation for Science, Technology & Research**, featuring official university branding and a professional workspace UI.

## 🛠️ Technology Stack
- **Frontend**: React, TailwindCSS, Lucide Icons, Framer Motion
- **Backend**: FastAPI (Python), SQLAlchemy, SQLite
- **AI Orchestration**: Groq (Llama 3.3 70B), Classical NLP
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
